# To play a video in terminal:

mpv --vo=tct <your_video_file>.mp4

1. Install Homebrew (macOS only)
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
2. Install mpv
   brew install mpv
